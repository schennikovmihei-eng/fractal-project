# Fractal Generator

Проект для генерации классических фракталов (Мандельброта и Жюлиа) на языке C. Отличается строгим соответствием стандартам качества: автоматическая сборка, проверка стиля кодирования и обширное использование отладочных проверок.

## Ключевые особенности

* Несколько фракталов: Генерация множества Мандельброта и множества Жюлиа с настраиваемыми параметрами.
* Автоматическая CI/CD сборка: Проект автоматически собирается и тестируется на GitHub Actions с использованием трёх систем сборки.
* Стиль кодирования Linux Kernel: Весь код отформатирован утилитой clang-format в соответствии со стилем ядра Linux (отступ 8 пробелов).
* Отладочный код: Повсеместное использование assert() для проверки входных параметров и инвариантов.
* Кросс-платформенность: Поддержка сборки через Make (для Windows и Linux), CMake и Meson.

## Требования

* Компилятор gcc или clang с поддержкой C11.
* Для локальной сборки: make, cmake и/или meson с ninja.
* Для проверки стиля кода: clang-format (конфигурация уже в проекте).

## Сборка и установка

Проект можно собрать тремя способами.

### 1. Сборка с помощью Make (рекомендуется для локальной разработки)
```bash
# Релизная сборка и запуск
make release
./bin/fractal

# Отладочная сборка с тестами
make debug
./bin/fractal_test

# Очистка артефактов сборки
make clean

### Сборка с помощью CMake

```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make
./fractal

### Сборка с помощью Meson

```bash
meson setup build --buildtype=release
meson compile -C build
./build/fractal

### Использование

После сборки запустите основную программу:
./bin/fractal  # или ./fractal, или ./build/fractal в зависимости от системы сборки
Программа автоматически сгенерирует в текущей директории три файла в формате PGM:

- mandelbrot.pgm — фрактал Мандельброта (бинарный формат P5).
- mandelbrot_ascii.pgm — фрактал Мандельброта (ASCII-формат P2).
- julia.pgm — фрактал Жюлиа (бинарный формат P5).
